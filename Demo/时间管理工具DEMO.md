我有一个想针对每天8小时的时间管理工具，定位主要是围绕下班后的8小时，记录时间和待办事项。思路如下：

1. 软件是在华为手机上的APP，界面风格符合鸿蒙应用开发要求
2. 所有功能可以分为：今日安排、新增事项、时间统计、个人设置四个。其中今日安排、新增事项、时间统计为界面下面的菜单栏，个人设置为隐藏式，主界面右上角有设置按钮，点击后浮出个人设置
3. 今日安排功能：通过刻度尺的形式，在界面左半侧，显示出空闲时间刻度、已经分配好任务，其中任务用不同颜色区分。界面右半侧，则是针对每个任务，显示出该任务的逐条待办。比如：21:00--22:00 任务：锻炼身体（绿色），右边对应显示1. 跑步30分钟2. 无氧20分钟 3. 拉伸10分钟
4. 新增事项功能：点击主界面下方菜单。中间的加号，可以新增两种类型，一个是任务，一个是事项。任务是时间块分配的大项，可以设置任务的标题、颜色。事项是在任务中需要做的具体的事，可以设置事项内容、优先级，但颜色与任务保持一致
4. 时间统计功能：可以用环形图结构图的方式，切换每月、每周的各任务的占用时间。也可以再追加一个柱形图或者折线图的方式，来统计任务时间，或者事项数量
4. 个人设置：管理登录账号（通常是手机或邮箱）、提醒、界面配色（深色、浅色）等基础设置。设置内容当即保存。
